1° Aritmeticos: Os operadores aritmeticos do javascript, assim como outras linguagens de progação são:

+ -> soma
- -> subtrai
* -> multiplica
/ -> divide
% -> divisão inteira
** -> exponenciação

Todos os operadores acima são chamados de binários, pois necessitam de dois operandos para realizar as sua operações. 

ATALHOS PARA ATRIBUIÇÃO DE VALOR:

+= -> Atribuição soma, também serve para fazer a concatenação quando a variavel é string

-= -> atribuição de Subtração

*= -> atribuição de multiplicação 

/= -> atribuição de divisão
