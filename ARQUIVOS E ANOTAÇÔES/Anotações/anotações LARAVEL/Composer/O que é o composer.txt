O Composer é um gestor de dependencias para PHP, eu seto a dependencia que meu projeto precisa e ele já baixa de forma automatica, se a dependencia depende de outro pacote, ele já baixa de forma automatica também. O composer revolucionou o php. Podemos utilizar o Composer para atualizar as versões que escolhermos, quando surge uma nova atualização, o composer já faz a instalação automatica do pacote para nós.
